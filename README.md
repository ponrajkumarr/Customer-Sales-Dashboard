# Customer-Sales-Dashboard
This dashboard explores sales performance and customer characteristics for an electronic shop using Power BI. It combines sales, customer, and product data to uncover revenue trends, best-selling products, and demographic patterns to assist in business planning.

# 🛒 Customer & Sales Dashboard – Power BI

## 📊 Project Overview
This project presents an interactive **Power BI dashboard** built using customer and sales data from an electronic shop.  
The dashboard helps analyze sales trends, customer demographics, product performance, and overall revenue contribution.

---

## 🎯 Objectives
- Understand **total sales performance** and profitability
- Identify **best-selling products**
- Analyze **customer demographics & purchasing patterns**
- Track **sales trends over time**
- Support business decision-making with data-driven insights

---

## 📂 Dataset Summary
The dashboard was built using **three source files**:
1. **Sales** – Transaction details (sales date, product, customer, amount, quantity)
2. **Products** – Product information (name, category, price)
3. **Customers** – Customer details (gender, age, location)

---

## 🧰 Tools & Skills Used
- **Power BI Desktop**
- Data Modeling (Star Schema)
- Power Query for Data Cleaning
- DAX (Measures & KPIs)
- Interactive Visuals (Slicers, Charts, Tables)

---

## 📈 Key Metrics & Insights
- ✅ **Total Sales & Revenue Trend**
- ✅ **Top Product Categories & Best-selling Items**
- ✅ **Customer Count by Region, Gender, Age Group**
- ✅ **Average Order Value**
- ✅ **Repeat vs. Unique Customers**
- ✅ **Seasonal / Monthly Sales Trend**

---

## 🔍 Data Model
The data model follows a **Star Schema**:
- **Fact Table:** Sales
- **Dimension Tables:** Customers, Products


---

## 🚀 How to Use
1. Download the `.pbix` report file.
2. Open using **Power BI Desktop**.
3. Interact with slicers for time period, product categories, customer segments, etc.
4. Explore visuals for sales & customer insights.
